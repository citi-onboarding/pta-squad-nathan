# Índice: #

(!) Importante: Salve o arquivo como .md para a formatação ser aplicada, exemplo: "meuArquivo.md".

  
# Use os seguintes comandos para formatar o documento: #

# Títulos:
• # Título 1 → Título grande;
• ## Título 2 → Título médio;
• ### Título 3 → Título pequeno.

# Estilos de Texto:
• **texto** → Negrito;
• _texto_ → Itálico;
• --- → Linha divisória;
• `texto` → Formatação de código inline;
• ```linguagem → Bloco de código, se você especificar a linguagem, ele adiciona a formatação automaticamente (ex: tsx, js, html), exemplo:

```tsx  
// seu código aqui  
```

(!) Obs.: se você não especificar a linguagem, ele também funciona, porém não aplicará formatação.
